📖 ImageFlipbook
Interactive Page-Turning Experiences for the Modern Web

Turn Static Images Into Living Books

ImageFlipbook is a powerful JavaScript library that transforms sequences of images into immersive, page-turning experiences. Perfect for digital manuscripts, catalogs, comic books, and more.

📱

Mobile Ready

Responsive design with touch gestures, swipe navigation, and mobile fullscreen support.

🎨

Dual-Page Mode

Smart dual-page display that adapts to screen size with single-page fallback on mobile.

⚡

Easy Integration

Simple API, zero dependencies, works with any image hosting. Just add script tag and go!

🔧

Customizable

Flexible theming, custom controls, fullscreen mode, and keyboard navigation support.
